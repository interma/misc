## Operate Hadoop 2 using pdsh/pdcp

### usage:
```
# install
install-hadoop2.sh -f
# uninstall
./uninstall-hadoop2.sh
# update configure and restart all
./configure-hadoop2.sh -f /opt/hadoop-2.7.3/etc/hadoop/hdfs-site.xml -r
```
